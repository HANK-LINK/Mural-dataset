# Data-for-PGMv2
This repository contains a large mural dataset. If you need to use our data, please cite our paper. For details, please see the readme.
To accommodate the needs of different users, we have divided the mural dataset into a Base version and a full version. Due to file size limitations required by GitHub, we have stored the specific data layers in a cloud drive. The specific cloud drive link and corresponding key are stored in the PDF document corresponding to that version.
